- حفظ التقدم تلقائيا على السيرفر

- قاعدة بيانات D1 لتخزين المستخدمين والتقدم

## API Endpoints

### Authentication
- `POST /api/auth/register` - تسجيل حساب جديد
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/auth/me` - التحقق من الجلسة
- `POST /api/auth/logout` - تسجيل الخروج

### Progress
- `POST /api/progress/save` - حفظ التقدم (يتطلب توكن)
- `GET /api/progress/load` - تحميل التقدم (يتطلب توكن)

### Content
- `GET /api/levels` - كل المساقات
- `GET /api/lesson/:levelId/:lessonId` - درس محدد
- `GET /api/glossary` - المعجم
- `GET /api/quizzes` - كل الاختبارات
- `GET /api/quiz/:levelId` - اختبار محدد
- `GET /api/stats` - الإحصائيات

## التقنيات
- **Backend**: Hono + TypeScript + Cloudflare D1
- **Frontend**: TailwindCSS + Vanilla JS
- **Platform**: Cloudflare Pages
- **Database**: Cloudflare D1 (SQLite)
- **Auth**: Custom token-based authentication
- **Icons**: FontAwesome 6.5.1
- **Font**: Tajawal (Google Fonts)

## التشغيل المحلي
```bash
npm install
npm run build
npx wrangler d1 migrations apply accounting-academy-db --local
npx wrangler pages dev dist --d1=accounting-academy-db --local --port 3000
```
