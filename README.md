# أكاديمية المحاسبة المالية

## نظرة عامة
منصة تعليمية عربية شاملة لتعلم المحاسبة المالية من الصفر إلى الاحتراف، تتضمن 8 مساقات و36 درس مع فيديوهات تعليمية من يوتيوب واختبارات تفاعلية.

## المساقات والدروس

### 1. المستوى المبتدئ (6 دروس)
- ما هي المحاسبة المالية؟
- المعادلة المحاسبية الأساسية
- الحسابات وأنواعها
- القيد المحاسبي المزدوج
- دفتر اليومية ودفتر الأستاذ
- ميزان المراجعة

### 2. المستوى المتوسط (5 دروس)
- قائمة الدخل
- الميزانية العمومية
- التسويات الجردية
- قائمة التدفقات النقدية
- المحاسبة عن المخزون

### 3. المستوى المتقدم (6 دروس)
- تحليل النسب المالية
- المعايير الدولية IFRS
- محاسبة الزكاة وضريبة القيمة المضافة
- القوائم المالية الموحدة
- محاسبة الأدوات المالية
- الاعتراف بالإيراد (IFRS 15)

### 4. محاسبة التكاليف (5 دروس)
- مقدمة في محاسبة التكاليف
- تصنيف التكاليف
- نظام تكاليف الأوامر الإنتاجية
- نظام تكاليف المراحل الإنتاجية
- تحليل التعادل

### 5. المحاسبة الإدارية (4 دروس)
- مقدمة في المحاسبة الإدارية
- الموازنات التقديرية
- تحليل الانحرافات
- اتخاذ القرارات قصيرة الأجل

### 6. المراجعة والتدقيق (4 دروس)
- مقدمة في المراجعة والتدقيق
- إجراءات التدقيق والأدلة
- تقييم المخاطر والرقابة الداخلية
- الغش والأخطاء المحاسبية

### 7. المحاسبة الضريبية (3 دروس)
- مقدمة في النظام الضريبي
- ضريبة القيمة المضافة - تطبيق عملي
- الفوترة الإلكترونية (فاتورة)

### 8. المستوى الاحترافي (3 دروس)
- الشهادات المهنية في المحاسبة
- البرامج المحاسبية
- تمارين شاملة ومراجعة نهائية

## الميزات

### المحتوى التعليمي
- 36 درس مع محتوى تفاعلي غني
- فيديو يوتيوب تعليمي لكل درس
- 6 اختبارات تفاعلية مع تصحيح فوري
- 30 مصطلح محاسبي في المعجم (عربي-إنجليزي)

### الأدوات الحسابية (8 آلات حاسبة)
- حاسبة الإهلاك (القسط الثابت)
- حاسبة نقطة التعادل
- حاسبة النسبة الجارية
- حاسبة ضريبة القيمة المضافة
- حاسبة الزكاة
- حاسبة هامش الربح
- حاسبة العائد على حقوق الملكية (ROE)
- حاسبة النسبة السريعة

### تجربة المستخدم
- الوضع الداكن/الفاتح
- بحث شامل في الدروس والمصطلحات
- ملاحظات شخصية لكل درس
- حفظ التقدم والمفضلة تلقائياً
- تصدير التقدم كملف JSON
- اختصارات لوحة المفاتيح (/ للبحث)
- زر العودة للأعلى (Scroll-to-Top)
- تأثيرات حركية سلسة (Animations)
- تصميم متجاوب لجميع الأجهزة

### صفحات إضافية
- نصائح مهنية للمحاسبين
- موارد وروابط مفيدة (IFRS, SOCPA, هيئة الزكاة)
- صفحة "عن الأكاديمية"
- Footer احترافي مع روابط التنقل

## API Endpoints
- `GET /api/levels` - كل المساقات
- `GET /api/lesson/:levelId/:lessonId` - درس محدد
- `GET /api/glossary` - المعجم
- `GET /api/quizzes` - كل الاختبارات
- `GET /api/quiz/:levelId` - اختبار محدد
- `GET /api/stats` - الإحصائيات

## التقنيات
- **Backend**: Hono + TypeScript
- **Frontend**: TailwindCSS + Vanilla JS
- **Platform**: Cloudflare Pages
- **Icons**: FontAwesome 6.5.1
- **Font**: Tajawal (Google Fonts)

## التشغيل المحلي
```bash
npm install
npm run build
npx wrangler pages dev dist --port 3000
```
