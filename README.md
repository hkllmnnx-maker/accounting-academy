# أكاديمية المحاسبة المالية v3.0

## نظرة عامة
منصة تعليمية عربية شاملة لتعلم المحاسبة المالية من الصفر إلى الاحتراف، تتضمن 8 مساقات و46 درس مع فيديوهات تعليمية من يوتيوب واختبارات تفاعلية ونظام تسجيل دخول لحفظ التقدم.

## المساقات والدروس (46 درس)

- ما هي المحاسبة المالية؟
- المعادلة المحاسبية الأساسية
- الحسابات وأنواعها
- القيد المحاسبي المزدوج
- دفتر اليومية ودفتر الأستاذ
- ميزان المراجعة
- **المبادئ المحاسبية الأساسية** (جديد)
- **الدورة المحاسبية الكاملة** (جديد)

### 2. المستوى المتوسط (7 دروس)
- قائمة الدخل
- الميزانية العمومية
- التسويات الجردية
- قائمة التدفقات النقدية
- المحاسبة عن المخزون
- **الأصول الثابتة والإهلاك** (جديد)
- **قائمة التغيرات في حقوق الملكية** (جديد)

### 3. المستوى المتقدم (7 دروس)
- تحليل النسب المالية
- المعايير الدولية IFRS
- محاسبة الزكاة وضريبة القيمة المضافة
- القوائم المالية الموحدة
- محاسبة الأدوات المالية
- الاعتراف بالإيراد (IFRS 15)
- **عقود الإيجار (IFRS 16)** (جديد)

### 4. محاسبة التكاليف (6 دروس)

- تصنيف التكاليف
- نظام تكاليف الأوامر الإنتاجية
- نظام تكاليف المراحل الإنتاجية
- تحليل التعادل
- **التكاليف المعيارية** (جديد)

### 5. المحاسبة الإدارية (5 دروس)
- مقدمة في المحاسبة الإدارية
- الموازنات التقديرية
- تحليل الانحرافات
- اتخاذ القرارات قصيرة الأجل
- **تسعير المنتجات والخدمات** (جديد)

### 6. المراجعة والتدقيق (5 دروس)
- مقدمة في المراجعة والتدقيق
- إجراءات التدقيق والأدلة
- تقييم المخاطر والرقابة الداخلية
- الغش والأخطاء المحاسبية
- **أخلاقيات المهنة والمسؤولية المهنية** (جديد)

### 7. المحاسبة الضريبية (4 دروس)
- مقدمة في النظام الضريبي
- ضريبة القيمة المضافة - تطبيق عملي
- الفوترة الإلكترونية (فاتورة)
- **التخطيط الضريبي** (جديد)

### 8. المستوى الاحترافي (4 دروس)

- البرامج المحاسبية
- تمارين شاملة ومراجعة نهائية
- **المحاسبة في عصر التحول الرقمي** (جديد)

## الميزات الجديدة v3.0

### نظام تسجيل الدخول وحفظ التقدم
- تسجيل حساب جديد (اسم مستخدم + بريد إلكتروني + كلمة مرور)
- تسجيل الدخول والخروج
- حفظ التقدم تلقائيا على السيرفر (لا يضيع حتى لو مسحت بيانات المتصفح)
- مزامنة التقدم بين الأجهزة المختلفة
- قاعدة بيانات D1 لتخزين المستخدمين والتقدم

### المحتوى التعليمي
- 46 درس مع محتوى تفاعلي غني (زيادة 10 دروس)
- فيديو يوتيوب تعليمي لكل درس
- 6 اختبارات تفاعلية مع تصحيح فوري
- 32 مصطلح محاسبي في المعجم (عربي-إنجليزي)
- 6 آلات حاسبة محاسبية متخصصة

## API Endpoints

### Authentication
- `POST /api/auth/register` - تسجيل حساب جديد
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/auth/me` - التحقق من الجلسة
- `POST /api/auth/logout` - تسجيل الخروج

### Progress
- `POST /api/progress/save` - حفظ التقدم (يتطلب توكن)
- `GET /api/progress/load` - تحميل التقدم (يتطلب توكن)

### Content
- `GET /api/levels` - كل المساقات
- `GET /api/lesson/:levelId/:lessonId` - درس محدد
- `GET /api/glossary` - المعجم
- `GET /api/quizzes` - كل الاختبارات
- `GET /api/quiz/:levelId` - اختبار محدد
- `GET /api/stats` - الإحصائيات

## التقنيات
- **Backend**: Hono + TypeScript + Cloudflare D1
- **Frontend**: TailwindCSS + Vanilla JS
- **Platform**: Cloudflare Pages
- **Database**: Cloudflare D1 (SQLite)
- **Auth**: Custom token-based authentication
- **Icons**: FontAwesome 6.5.1
- **Font**: Tajawal (Google Fonts)

## التشغيل المحلي
```bash
npm install
npm run build
npx wrangler d1 migrations apply accounting-academy-db --local
npx wrangler pages dev dist --d1=accounting-academy-db --local --port 3000
```
